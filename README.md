# Rossmann Store Sales Prediction

A sales prediction system built using **Linear Regression** and **Random Forest** to forecast store sales, trained on the [Rossmann Store Sales dataset]((https://www.kaggle.com/competitions/rossmann-store-sales/data?select=train.csv)) from Kaggle.

## Overview

This project explores and compares two regression approaches — Linear Regression and Random Forest — to predict daily sales for Rossmann stores based on historical sales data, store information, and promotional activity.

## Tech Stack

- Python
- scikit-learn
- pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset

The [Rossmann Store Sales](https://www.kaggle.com/competitions/rossmann-store-sales/data?select=train.csv) dataset from Kaggle, containing historical sales data for 1,115 Rossmann stores.

## Models Used

- **Linear Regression** — baseline model for sales prediction
- **Random Forest Regressor** — ensemble model for improved accuracy

## How to Run

1. Clone the repo
2. Download the dataset from Kaggle and place it in the project directory
3. Open the notebook in Google Colab or Jupyter
4. Run the cells sequentially
